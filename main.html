My Journey into Frontend Development

On April 16th, I officially began my 100 Days of Code journey. The excitement of writing my first lines of JavaScript reminded me why I chose this path. Every day brings new challenges, but also growth.

Understanding HTML & CSS

HTML provides the structure, and CSS brings style. I’m learning how to use semantic HTML tags properly, making my web pages more accessible and meaningful. CSS is teaching me how design and usability go hand in hand.

JavaScript – The Real Game Changer

JavaScript allows me to add interactivity to my projects. Whether it is form validation, animations, or dynamic content, JS is the engine behind the frontend experience. I can not wait to dive deeper into DOM manipulation and frameworks like React.
